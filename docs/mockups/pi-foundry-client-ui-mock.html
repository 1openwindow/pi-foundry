Foundry-first · pi-based agent client

Pi Foundry Client — user task workspace + agent studio

A complete UI mock for a product that maximally reuses Microsoft Foundry for hosting, deployment, identity, and runtime control, while adding a product layer for AgentDefinition, Release, Binding, Run, and Artifact. The user experience is centered on tasks, long-running execution, resumable sessions, and downloadable outcomes.
