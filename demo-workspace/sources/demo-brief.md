# Demo Brief: Coding Agent Competitive Analysis

## User request

分析 Claude Code、OpenAI Codex、OpenCode 三个 coding agent 的优势和短板，做成一个带声音的汇报材料。

## Target audience

Technical leadership:

- CTO
- VP Engineering
- Engineering managers
- Platform engineering leads

## Desired artifact

A Hyperframes HTML presentation with Chinese narration audio. Do not render video.

Output directory:

```text
/files/coding-agent-comparison/
```

Expected files:

```text
index.html
narration.mp3
script.md
outline.md
README.md
```

Optional:

```text
narration.vtt
```

## Recommended story arc

1. Why coding agents matter now
2. One-line positioning of each agent
3. Claude Code: best for interactive local engineering
4. OpenAI Codex: best for cloud task delegation and issue-to-PR workflows
5. OpenCode: best for self-hosted/extensible internal platforms
6. Comparison matrix across enterprise evaluation dimensions
7. Adoption recommendations by user group
8. Final decision framework: buy, adopt, build, or combine

## Tone

- Executive-friendly
- Clear and balanced
- Avoid hype
- Distinguish facts, assumptions, and recommendations
- Emphasize governance, cost, risk, and integration needs

## Visual direction

Use a clean corporate style:

- 16:9 layout
- dark navy or light neutral background
- strong title hierarchy
- comparison cards
- score matrix
- restrained motion
- readable Chinese text
- no video render required

## Narration direction

Chinese voiceover:

- Voice: `zh-CN-XiaoxiaoNeural`
- Tone: professional, concise, strategic
- Duration target: 2-3 minutes
- Prefer short sentences
- Explain the decision logic, not every bullet on screen
